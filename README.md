## Setup
Create project with virtual environment

```console
$ mkdir myproject
$ cd myproject
$ python3 -m venv venv
```

Activate it
```console
$ . venv/bin/activate
```

or on Windows
```console
venv\Scripts\activate
```

Install Dependencies
```console
$ pip install -r requirements.txt
```

Run the app
```console
$ flask run
```

## Admin functionality
- View all events
- Add a new event
- Update event details
- Delete event

## User functionality
- View available events
- Book tickets for an event
- Cancel booking
